= Swatch

A simple command line utility in Ruby to track your time.
Can be use with todo.txt task list.

== Usage

<tt>swatch command [option] arg</tt>

Going in a task :
<tt>swatch in My awesome task</tt>

Getting out:
<tt>swatch out</tt>

Timing a todo.txt task:
<tt>swatch in -t 42</tt>
